# raspigibbon_sim

## About

Raspberry Pi Gibbon Simulator suites runs on Gazebo

## Requirements

requires the following to run:

* Ubuntu
  * Ubuntu Trusty 14.04
* ROS
  * ROS Indigo
* Gazebo
  * Gazebo 2.x

or

* Ubuntu
  * Ubuntu Trusty 16.04
* ROS
  * ROS Kinetic
* Gazebo
  * Gazebo 7.x
